STUDENT WORKSHOP

The goal of this workshop is to allow students an opportunity to present ongoing work and interact with other researchers in order to get feedback, exchange experiences, make contacts, and learn about related work. Students not only will be able to learn about other students’ work but also interact with well established researchers in the field who will participate in the workshop as mentors and organizers. We are soliciting two-page extended abstracts on the same topics as covered in the original CFP of PAM 2009. Submissions will be reviewed by the program committee, from which a number of abstracts will be selected for inclusion in the workshop program. Authors will present their work as a poster.

SUBMISSION INFORMATION
Submissions must follow ACM guidelines: double column, with each column 9.25" by 3.33", 0.33" space between columns and must be submitted in PDF format. Use the paper submission site to register and submit a paper for review.
If using LaTex, the following style file must be used for submission of papers:
  • PAM2009-submission.cls
  • PAM2009-sample.tex
  • PAM2009-sample.pdf
These are modified versions of the WWW2009 poster style files. More information on the style files can be found at:
http://www2009.org/submission.html


Posters should have the following format: A0 size (841x1189 mm).
PAM 2009 will provide boards for presenting posters. Authors are responsible for printing and mounting their posters.
